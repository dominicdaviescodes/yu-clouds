About Me.

I'm your typical tech guy, you know the one who works in IT and makes websites for friends and family? I used just about every tool I could until I found it really limiting not knowing how to code.

I had a bit of a head start just from using WordPress and my IT background has come in handy especially with debugging code and searching for solutions to problems.

Not knowing any better I began learning to code in 2016 doing dozens of Ruby on Rails tutorials, its still my first love. I thought I was going to make the next Twitter!

I obviously lacked the basics back then but I was having so much fun making clones of Twitter and Pinterest using the command line, git and github for version control before deploying them to Heroku.

Then the pandemic happened. And here I am looking for a developer job. You can follow my weekly learning journal over at LinkedIn.
